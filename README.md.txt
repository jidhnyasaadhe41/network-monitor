
# Network Monitoring Dashboard

This is a simple full stack project to monitor basic network logs and detect anomalies.

## Tech Used
- React (Frontend)
- Node.js + Express (Backend)
- Python (for anomaly detection)

## Features
- Displays network status and latency
- Simple API integration
- Basic anomaly detection script

## How to Run

### Backend
cd backend  
npm install  
node server.js  

### Frontend
cd frontend  
npm install  
npm start  

### Python Script
cd ai  
python anomaly.py  

## Notes
This is a beginner-level project built to understand full stack flow and basic monitoring concepts.
